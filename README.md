# Gadget Store

Gadget Store is a **frontend-first e-commerce web application** built using **Next.js 15/16 (App Router)**.
The project focuses on clean routing, reusable layouts, and mock authentication, with future support for an Express.js backend.

---

## Features

* Public landing page with 7 sections
* Reusable Navbar and Footer
* Mock login using email and password
* Public gadget listing page
* Dynamic gadget details page
* Protected add gadget page (planned)
* Responsive UI with Tailwind CSS

---

## Technologies Used

* Next.js 15/16 (App Router)
* JavaScript (No TypeScript)
* Tailwind CSS
* Express.js (planned)
* Cookie-based authentication (mock)
